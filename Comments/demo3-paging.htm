Paging Demos

This is a demo of paging through data.

The "Query String" button calls code that demonstrates how a typical developer might implement paging today.

The "Queryable" button turns it up a notch and shows paging using a service method that returns an IQueryable. Note that the client code hasn't changed much at all, but the server code is much simpler.
